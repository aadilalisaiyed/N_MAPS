ğŸ“– Overview
Sentinel's Vault is a resilient, offline communication ecosystem designed for disaster-stricken areas where traditional cellular and internet infrastructure has failed.

By leveraging a decentralized ESP32 Mesh Network, this system creates a self-healing grid that allows victims and rescue teams to exchange critical information (location, status, resource needs). Unlike standard mesh networks, Sentinel's Vault integrates hardware-level encryption to ensure that sensitive data remains secure, preventing unauthorized interception during critical operations.

âš™ï¸ Architecture
Node A (Victim/Field Unit): Broadcasts encrypted distress signal containing UUID and GPS data.

The Mesh: Intermediate nodes relay the packet via the most efficient path using the mesh routing algorithm.

Node B (Command Center/Gateway): Receives the encrypted packet, decrypts it using the hardware key, and logs the data to the central dashboard.
